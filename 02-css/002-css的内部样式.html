海内外舆论高度关注习近平调研央媒

中共中央总书记、国家主席、中央军委主席习近平今天在北京主持召开党的新闻舆论工作座谈会并发表重要讲话。为召开这次座谈会，习近平到人民日报社、新华社、中央电视台进行了实地调研。消息受到海内外舆论高度关注，网友也纷纷通过社交平台为总书记点赞。

“习近平通过新媒体致元宵节问候”“习近平为新闻工作者点赞”“习近平体验《新闻联播》演播室”等消息，19日上午第一时间登陆各大新闻媒体和门户网站的头条位置。美国华尔街日报网站、新加坡联合早报网、香港南华早报网、台湾“中央社”、台湾旺报网站、中时电子报等海外媒体也重点报道了习近平此次调研活动。香港大公网、文汇网、星岛环球网均在重要位置展示活动的现场影像。

香港文汇网转引媒体评论文章指出，这并不是习近平首次亲临媒体。他曾多次谈及新闻媒体领域的问题。文章回顾了习近平与媒体的数次“亲密接触”，并提到“长期以来，总书记一直很重视媒体工作”。
